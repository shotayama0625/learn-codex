# 冷蔵庫レシピ提案アシスタント

冷蔵庫の食材から「作れる料理」を OpenAI API で提案するシンプルなWebアプリです。

## 対応機能

- 冷蔵庫にある食材から料理提案（OpenAI API 利用）
- API Key の直接入力に加え、`.txt` / `.env` ファイル読み込みに対応
- 和食 / 洋食 / 中華でのジャンル絞り込み
- かんたん / ふつう / むずかしいでの難易度絞り込み
- 作った料理の登録（料理名から食材を OpenAI API で推定して保存）
- 登録済みの料理が提案候補にある場合は優先表示

## 使い方

1. `index.html` をブラウザで開きます。
2. OpenAI API Key を直接入力、または API Key ファイル（1行目にキー / `OPENAI_API_KEY=...` 形式）を指定し、食材を入力します。必要に応じてジャンル・難易度を指定します。
3. 「作れる料理を提案する」を押します。
4. 作った料理は料理名を入力して「作った料理を登録する」で保存します。

> 注意: API キーはブラウザ内でのみ使用し、ローカルストレージには保存しません。ファイル指定時も読み込みのみで保存しません。
> `api-key.example.env` をコピーして自分のキーを書き、ファイル選択で読み込めます。

## ローカル起動（任意）

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。
